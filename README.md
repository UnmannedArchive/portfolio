# Portfolio

Personal site for **Joseph Solomon** — Next.js (App Router), Tailwind CSS, Framer Motion.

## Folder

This app lives only in **`portfolio/`** at the repo root. Treat it as its own project for tooling and deploys.

## Local dev

```bash
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Create a new Vercel project named **`portfolio`** (or rename an existing one to that in **Project → Settings → General**).
2. **If this folder is inside a larger Git repo** (siblings like `aiflix/`, `j-feed/`, etc.), set **Root Directory** to **`portfolio`** so Vercel builds only this app.
3. **If you use a Git repo that contains only this app**, connect that repo and leave the root at `.`.

The npm package name is **`portfolio`** (`package.json`).
